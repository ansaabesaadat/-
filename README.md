# MyNasab Clone — Syed/Sadat Family Tree Website

A pixel-faithful clone of **mynasab.com** — the Syed & Sadat genealogy website — built as a single `index.html` file ready for GitHub Pages.

## 🚀 Deploy to GitHub Pages (3 steps)

1. Create a new GitHub repository (e.g. `mynasab-clone`)
2. Upload `index.html` to the root
3. Go to **Settings → Pages → Source: main branch → /root** → Save

Your site will be live at:  
`https://YOUR_USERNAME.github.io/mynasab-clone/`

## 📁 File Structure
